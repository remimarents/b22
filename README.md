# b22

Et morsomt retro-nettleserspill for miljoeterapeuter, primart laget for mobil.
Spillet er laget for Barum kommune, men andre kan gjerne lage sin egen versjon
ved a bytte logoen pa jakken i figur-grafikken.

## Kjor lokalt

1. Start en enkel webserver i prosjektmappen:
   - `python3 -m http.server 8080`
2. Apne:
   - `http://localhost:8080`

## Produksjon

Spillet er publisert under:

- `https://terapisomvirker.no/b22/`

## Teknologi

- HTML/CSS
- JavaScript
- Phaser 3
- Enkel PHP-endpoint for highscore
